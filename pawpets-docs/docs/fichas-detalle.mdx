# Fichas (Detalle)

Esta página muestra la **ficha completa** del animal seleccionado desde el catálogo.

## Qué información se muestra

- Imagen del animal
- Nombre y raza
- Edad, sexo, ubicación y tamaño
- Chip y número de chip
- Descripción
- Acciones: **Volver**, **Adoptar** y **Favorito**
